GetAround — Analyse & Prédiction de Prix

Dashboard interactif d'analyse tarifaire sur la plateforme de location de voitures GetAround. Exploration des données, identification des facteurs de prix et modèle de prédiction de loyer journalier.

🎯 Contexte & Objectif

GetAround est la version européenne d'Airbnb pour les voitures. Le projet avait pour but d'analyser les facteurs influençant le prix de location journalier des véhicules, puis de construire un modèle de Machine Learning permettant de recommander un prix optimal à un propriétaire.

🛠️ Méthodologie

1.
Exploration & nettoyage — Analyse des distributions, gestion des valeurs manquantes, outliers sur le prix journalier.
2.
Feature Engineering — Encodage des variables catégorielles (marque, modèle, carburant), normalisation.
3.
Modélisation — Comparaison LinearRegression, Ridge, RandomForest. Sélection par validation croisée.
4.
Dashboard Streamlit — Visualisations interactives Plotly + UI de prédiction de prix à la demande.
