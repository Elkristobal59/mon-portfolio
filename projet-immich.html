INFRASTRUCTURE — HOMELAB

Cloud Photo Privé — Immich

Déploiement d'une stack Docker sur serveur Lenovo (Proxmox Virtualization) avec stockage distant sur un NAS Synology DS218 Play, accessible depuis n'importe où via le réseau privé Tailscale (WireGuard). Sauvegarde automatique de 2 iPhones — sans ouvrir un seul port.

Docker Compose Synology NAS Immich PostgreSQL (pgvecto-rs) Redis cache Intel QuickSync (HWA) Tailscale VPN WireGuard

🎯 Pourquoi ce projet ?

Après avoir utilisé Google Photos et iCloud, j'ai voulu reprendre le contrôle de mes données personnelles. L'objectif était de construire une solution de cloud privé complète, sécurisée, et accessible depuis n'importe où dans le monde — sans abonnement mensuel et sans compromettre la sécurité.

📦 Services Déployés

📸
Immich & Machine Learning
Photo, Vidéo & IA

Alternative open-source à Google Photos. Reconnaissance faciale et d'objets propulsée par l'intelligence artificielle. Initialement indexé via GPU (MSI) pour traiter 200 000+ photos, puis migré sur Lenovo pour la maintenance courante.

🗄️
PostgreSQL & Redis
Base de données & Cache

Base de données robuste avec l'extension pgvecto-rs pour gérer efficacement la recherche vectorielle (embeddings des photos), épaulée par un broker Redis pour le système de file d'attente (traitement vidéo intensif, ML).

🔐
Tailscale (WireGuard)
VPN Mesh

Réseau privé mesh basé sur WireGuard. Chaque appareil rejoint le réseau Tailscale et accède directement au NAS sans ouvrir de port, avec chiffrement de bout en bout et MagicDNS.

💡 Apprentissages Clés

  • ✓ Tailscale (basé sur WireGuard) offre un VPN mesh plug-and-play qui sécurise l'accès distant sans aucune configuration firewall.
  • ✓ Docker Compose simplifie drastiquement la gestion des dépendances multi-services (bases de données, cache Redis, workers...).
  • ✓ La gestion des volumes Docker pour la persistance des données est critique — une mauvaise config = perte de données.
  • ✓ Immich a bénéficié de la puissance CUDA (MSI) pour l'indexation initiale massive, assurant un traitement rapide des visages et objets, avant de basculer sur une architecture stable CPU pour le quotidien.
  • ✓ Les architectures orientées micro-services pour l'IA nécessitent un stockage optimisé pour les vecteurs (extensions comme pgvecto-rs).
