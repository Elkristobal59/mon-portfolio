Étape 01
Scraping

Le script Python visite tldr.tech/data/YYYY-MM-DD, parse le HTML avec BeautifulSoup et extrait les articles en filtrant automatiquement les sponsors.

Python BeautifulSoup4 requests
🤖
Étape 02
Traduction & Catégorisation

Les articles bruts (EN) sont envoyés à l'API Gemini avec un prompt structuré. Le modèle retourne un tableau JSON avec titres traduits, résumés courts et catégories (News, Tool, Tutoriel, Deep Dive). Un Hot Swap bascule sur le modèle de secours si surcharge.

Gemini API JSON
⚙️
Étape 03
Publication automatique

Un workflow GitHub Actions déclenché chaque lundi et jeudi exécute le script, insère la nouvelle édition en tête du JSON (10 éditions max, FIFO), puis commit et push automatiquement vers GitHub Pages.

GitHub Actions GitHub Pages
Stack complète : Python 3.11 BeautifulSoup4 Gemini 3 Flash GitHub Actions GitHub Pages JSON
